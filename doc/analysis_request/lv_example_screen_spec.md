## 要求事項詳細
1. 画面の以下の仕様を教えてください。
  1.1. 画面のサイズ、配置座標、プロパティ
  1.2. GUI部品の大きさ、座標、プロパティ
  1.3. GUI部品にイベントが設定されていればイベントハンドラを教えてください。
  1.4. GUI部品のプロパティは初期値を書いてください。例えばボタンであればOFF状態、チェックボックスであればチェックなし、など。
2. 画面遷移を明確にしてください。
